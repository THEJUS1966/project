# **App Name**: Letter Perfect

## Core Features:

- Letter Generation: Generate letter content based on user input and selected tone using a generative AI tool.
- Tone Adjustment: Modify the generated letter based on a tone chosen by the user (friendly, professional, formal, polite, casual).
- Format Output: Formats the output by adding ‘From’ and ‘To’ address if the user provides them.
- Recipient Greetings: Provide multiple recipient options, e.g. ‘Sir’, ‘Madam’ or specific name for a personalised salutation, or ‘None’ for no greeting at all.

## Style Guidelines:

- Primary color: Calm light blue (#A7D1E7) to evoke a sense of professionalism and clarity.
- Background color: Light gray (#F0F4F7) for a clean, uncluttered interface.
- Accent color: Soft green (#90EE90) to highlight important elements and actions.
- Font: 'Inter' (sans-serif) for clear and modern readability in both headings and body text.
- Simple, minimalist icons to represent different tone options and formatting actions.
- Clean, well-spaced layout with a focus on readability and ease of use.
- Subtle transitions and animations to enhance user experience without being distracting.
